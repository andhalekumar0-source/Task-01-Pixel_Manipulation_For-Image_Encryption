# Task-01-Pixel_Manipulation_For-Image_Encryption
A simple Python tool that encrypts and decrypts images using pixel manipulation. It changes pixel RGB values with a secret key, making the image unreadable until decrypted. Uses Pillow and NumPy for secure, fast, and reversible image encryption.
